## Descripción
Este programa permite registrar estudiantes con sus calificaciones, calcular su promedio, determinar quién obtuvo el mejor promedio y guardar los resultados en un archivo `resultados.txt`.

## Requisitos

- Python 3.x
- Librerías:

  - pandas
  - seaborn
